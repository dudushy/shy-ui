# shy-ui

A professional Angular 21 UI component library and design system.

## Packages

- **`@shy/ui`** (`projects/shy-ui`) — Reusable UI component library
- **`showcase`** (`projects/showcase`) — Interactive component showcase app

## Getting Started

```bash
npm install
npm run build:lib   # Build the library
npm start           # Serve the showcase app
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm start` | Serve the showcase app |
| `npm run build` | Build library + showcase |
| `npm run build:lib` | Build the shy-ui library |
| `npm run build:app` | Build the showcase app |
| `npm run serve` | Serve showcase (opens browser) |

## Components

- **Button** — Primary, secondary, outline, ghost, danger variants with loading state
- **Card** — Container with optional elevation and padding variants
- **Badge** — Colored status labels
- **Input** — Form input with label, hint, error state, and CVA support
- **Modal** — Overlay dialog with header, body and footer slots
- **Spinner** — Loading indicator
- **Icon** — SVG icon set with 30+ icons
- **EmptyState** — Empty content placeholder
